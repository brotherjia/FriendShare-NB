好友与屏蔽共享 NB优化版V1.1 使用说明
====================================

一、插件用途
------------

同步同服务器、同阵营角色的好友列表和屏蔽列表。

NB优化版支持跨账号好友同步，并可避免把自己的小号/同步角色占用好友栏位。


二、安装
--------

1. 将 FriendShare 文件夹放入：

   World of Warcraft\Interface\AddOns\FriendShare

2. 角色选择界面点击“插件”，启用“好友与屏蔽共享 NB优化版V1.1”。

3. 游戏已经打开时，输入：

   /reload


三、同账号同步
--------------

同账号角色不需要配置。

依次登录各角色，插件会自动读取并合并好友数据。以后添加或删除好友，其他角色登录时会自动同步。


四、跨账号同步
--------------

跨账号同步要求两个角色同时在线，并且双方互相添加为同步角色。

以 Nbpala 和 Nbmagedad 为例：

1. Nbpala 输入：

   /fs peer add nbmagedad

2. Nbmagedad 输入：

   /fs peer add nbpala

3. 双方检查：

   /fs peers

4. 首次同步或补同步：

   /fs sync

完成后，两个同步角色同时在线时，添加/删除好友会自动同步。


五、小号删除开关
----------------

默认会自动移除好友列表里的小号/同步角色。

保留自己的小号好友：

   /fs removealts off

重新开启自动删除：

   /fs removealts on

这个开关会同步给同账号角色和在线的跨账号同步角色。离线的跨账号角色之后双方在线执行 /fs sync，也会同步到较新的状态。


六、常用命令
------------

/fs help
显示帮助。

/fs peers
查看跨账号同步角色。

/fs peer add 角色名
添加跨账号同步角色。

/fs peer remove 角色名
移除跨账号同步角色。

/fs sync
与全部同步角色交换数据。

/fs sync 角色名
只与指定角色交换数据。

/fs removealts on
开启自动删除小号/同步角色，并立即检查当前好友列表。

/fs removealts off
关闭自动删除小号/同步角色。

/fs import
手动导入已保存好友。

/fs reset
用当前好友列表重置全局好友记录。请谨慎使用。


七、屏蔽列表命令
----------------

/is help
显示屏蔽列表帮助。

/is import
导入已保存屏蔽列表。

/is reset
用当前屏蔽列表重置全局屏蔽记录。


八、注意事项
------------

1. 跨账号同步只支持同服务器、同阵营。

2. 跨账号同步前，双方都要安装同版本插件，并互相执行 /fs peer add。

3. 好友列表仍受 WLK 3.3.5 客户端上限限制，满员时无法继续导入。

4. 如果跨账号同步没有生效，请确认双方在线、/fs peers 名字正确，然后重新输入 /fs sync。
