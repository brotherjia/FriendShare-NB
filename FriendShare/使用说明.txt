好友与屏蔽共享 NB优化版V1.0 使用说明
====================================

一、插件用途
------------

本插件用于在同服务器、同阵营的角色之间同步好友列表和屏蔽列表。

NB优化版增加了跨账号好友同步功能。配置好同步角色后，一个角色添加或删除好友，其他已同步的角色也会自动添加或删除。

插件会尽量避免把你自己的角色、小号、同步角色添加进好友列表，节省好友栏位。


二、安装方法
------------

1. 将 FriendShare 文件夹放入魔兽世界插件目录：

   World of Warcraft\Interface\AddOns\FriendShare

2. 进入游戏角色选择界面，点击左下角“插件”，确认“好友与屏蔽共享 NB优化版V1.0”已启用。

3. 如果游戏已经打开，请在每个角色上输入：

   /reload

   或者完全退出游戏后重新登录。


三、同账号角色同步
------------------

同一个账号下的角色不需要额外配置。

你只需要依次登录这些角色，插件会自动读取并导入全局好友列表。

例如：

1. 登录角色 A，插件记录角色 A 的好友。
2. 登录角色 B，插件会自动把全局好友导入角色 B。
3. 之后角色 A 或角色 B 添加、删除好友，其他角色登录时会自动同步。


四、跨账号角色同步
------------------

跨账号同步需要两个账号的角色同时在线，并且双方都把对方添加为同步角色。

假设：

- 账号 1 的角色叫 Nbpala，已有完整好友列表。
- 账号 2 的角色叫 Nbmagedad，想同步 Nbpala 的好友列表。

操作步骤：

1. 两个账号同时登录，确保 Nbpala 和 Nbmagedad 都在线。

2. 在 Nbpala 上输入：

   /fs peer add nbmagedad

3. 在 Nbmagedad 上输入：

   /fs peer add nbpala

4. 两边都可以输入下面命令检查同步角色是否配置正确：

   /fs peers

   Nbpala 上应该看到 Nbmagedad。
   Nbmagedad 上应该看到 Nbpala。

5. 在任意一边输入：

   /fs sync

   推荐在空好友列表的角色上输入，例如 Nbmagedad。

6. 同步成功后，Nbmagedad 会导入 Nbpala 的好友列表。


五、日常使用
------------

首次配置完成后，平时一般不需要再输入命令。

当两个同步角色同时在线时：

- 在任意一个角色上添加好友，其他同步角色会自动添加。
- 在任意一个角色上删除好友，其他同步角色会自动删除。

当其他角色不在线时：

- 当前账号会保存变更。
- 其他角色之后登录时会自动导入已保存的好友列表。
- 如果跨账号角色错过了实时同步，可以两个账号同时在线后手动输入：

  /fs sync


六、常用命令
------------

/fs help
显示帮助信息。

/fs peers
查看当前配置的跨账号同步角色。

/fs peer add 角色名
添加一个跨账号同步角色。

示例：

  /fs peer add nbpala

/fs peer remove 角色名
移除一个跨账号同步角色。

示例：

  /fs peer remove nbpala

/fs sync
与所有已配置的同步角色交换好友数据。

/fs sync 角色名
只与指定同步角色交换好友数据。

示例：

  /fs sync nbpala

/fs import
手动导入当前已保存的全局好友列表。

/fs reset
用当前角色的好友列表重置全局好友列表。

注意：reset 会覆盖当前服务器、当前阵营的全局好友记录，请谨慎使用。


七、屏蔽列表命令
----------------

/is help
显示屏蔽列表同步帮助。

/is import
导入全局屏蔽列表。

/is reset
用当前角色的屏蔽列表重置全局屏蔽列表。


八、注意事项
------------

1. 跨账号同步要求两个角色同时在线。

2. 同步范围按“服务器-阵营”区分。
   不同服务器、不同阵营不会混到同一份好友列表。

3. 好友上限仍受游戏客户端限制。
   WLK 3.3.5 默认好友上限通常为 50 人。如果好友满了，插件无法继续添加更多好友。

4. 自己的角色、小号、同步角色会被记录用于排除，但不会主动加入好友列表。
   如果旧版本已经把自己的角色加进好友列表，可以手动删除一次，插件不会再自动加回来。

5. 如果跨账号同步没有立刻生效，请确认：

   - 两个角色都在线。
   - 两边都安装并启用了同一个版本的插件。
   - 两边都已经互相执行过 /fs peer add。
   - /fs peers 显示的角色名没有拼错。
   - 重新输入 /fs sync。

6. 如果聊天框出现 FSHARE: 开头的密语，这是插件的兼容同步消息。
   正常情况下插件会尝试隐藏它；如果你的客户端无法隐藏，也不影响同步功能。


九、推荐首次配置流程
--------------------

以 Nbpala 和 Nbmagedad 为例：

1. 两个账号都更新插件并 /reload。
2. 同时登录 Nbpala 和 Nbmagedad。
3. Nbpala 输入：

   /fs peer add nbmagedad

4. Nbmagedad 输入：

   /fs peer add nbpala

5. 两边都输入：

   /fs peers

6. Nbmagedad 输入：

   /fs sync

7. 确认好友列表已同步。

完成后，以后正常添加或删除好友即可。
